# Cornbear Café Receipt (Java)

This Java program generates a detailed café receipt for Cornbear's Café at the University of Washington. It calculates drink subtotals, tax, fees, and the final total, then prints everything in a nicely formatted layout.

## Features
- Calculates subtotals for espresso, sesame lattes, and Seattle Fog drinks  
- Computes tax and service fees  
- Generates a custom “Cornbear fee”  
- Rounds values for currency formatting  
- Prints the total receipt with a timestamp  
- Uses arithmetic operators, modulo, type casting, and variables

## Skills Practiced
- Variables & data types  
- Arithmetic expressions  
- Rounding with casting  
- String formatting  
- Printing structured output
